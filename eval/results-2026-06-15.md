# A/B Result — 2026-06-15 (first real run)

**This is the first actual execution of the orchestration A/B.** It is run on the
**seed fixture only** (2 verify tasks × 3 planted defects a/b/c, + 1 divergent task),
so **n is far too small for statistical significance** (the pre-registered p<0.05 rule
is not satisfiable here — see `README.md` power note). Read this as a *mechanism /
direction probe*, not a benchmark.

## Setup (best available models, as requested)

- **Workers (held fixed, structure ablated):** Claude **Opus** (headline) and **Sonnet**
  (the model-swap placebo discriminator). Run via the Workflow tool.
- **Judge (independent, blind, non-Claude):** **GPT-5.2** (OpenAI direct), scoring each
  arm's defect list against the planted ground truth, blind to which arm produced it.
  16 verify-judge calls + 4 recall calls, 0 errors.
- **Arms:** `A` single mega-agent · `A2` prompt-matched (all lenses, one context; isolates
  the lens taxonomy) · `A_N` draw-matched (5 generic draws, union; isolates draw count) ·
  `B` panel (5 independent lens skeptics). 60 worker agents total.

## Honest reading

1. **The fixture is saturated (ceiling effect).** A single strong agent already catches
   essentially all planted defects (Opus single = 6/6; Sonnet single = 5/6). With recall at
   the ceiling there is almost no headroom for the panel to show a recall advantage — and it
   doesn't: **panel recall ≈ single recall on every stratum.**
2. **On cost, the panel is a clear LOSER here:** `caught_per_agent` is **~0.6 for the panel
   vs ~3.0 for a single agent** — the panel spends 5× the agents to catch the *same* defects.
   This cost disadvantage **persists across the Opus→Sonnet swap** (model-invariant ⇒
   structural/real), exactly the "cost regression" the project's own guardrails warned about.
3. **The two control arms did their job:** the draw-matched arm (`A_N`) has the **worst
   precision** (most false positives from 5 unfiltered draws) and even missed a c-defect on
   Opus — so "just take more draws" is *not* the win; the panel's lens-structure is cleaner
   than raw draws on precision (panel 0.455/0.604 > draw 0.40/0.419).
4. **The one faint positive:** where a single agent *missed* (Sonnet single missed a b-omission,
   recall 0.5), the **panel and the prompt-matched arm both caught it** — consistent with the
   hypothesis that decomposition helps omission-recall *when there is a miss to recover*. But
   this is a single instance (n=1), not a claim.

## What this means for T2 ("productivity demonstrably improves")

**Running the A/B did NOT ground T2 — on this fixture it points the other way.** The panel
showed **no recall gain and a 5× per-defect cost regression** on tasks a single strong agent
already handles. The honest, evidence-backed conclusion:

> The orchestration panel's value can only appear on **harder artifacts where a single pass
> actually misses defects** (recall below ceiling). This seed fixture is too easy to
> discriminate the arms. A positive T2 signal requires (a) a harder, partly
> independently-sourced fixture with subtle defects below the single-agent ceiling, and (b) a
> powered n. Until then, the measured direction is *null-to-negative for the panel on easy
> tasks*, which sharpens — rather than weakens — the project's honest posture: **scaffolding
> is not a free win; on easy work it is pure cost.**

This is the falsification discipline working as designed: the pre-registered harness was run,
and on this fixture it does **not** support a productivity claim.
