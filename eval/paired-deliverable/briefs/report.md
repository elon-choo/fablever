## 프로젝트
`b2b-coldmail` = 천기문 B2B 파트너스. 리워드·앱테크·콘텐츠 사업자의 공개 SNS 프로필을 발굴해 개인화 아웃리치를 보내고, 서포터즈 전용 링크의 클릭·결제·RS 50% 정산을 관리하는 내부 운영 콘솔. 운영 URL: b2b-coldmail.vercel.app.

## 이번 작업 라운드의 사실 (이것만 씁니다 — 없는 사실을 지어내지 마세요)
- 목표 3개였음: **GA**, **GB**, **GC**.
- **GA**: Step 1~2 완료. Step 2에서 **로그인을 브릭시키는 SQL 오류 발견**(`expires_at` 컬럼 참조가 모호해서 쿼리 실패). Step 3 이후는 오너 결정 대기로 멈춤.
- **GB**: 이슈 트리아지 완료 — **P2 등급 16건 확인**. 수정은 **미실행**(권한 게이트에 걸림).
- **GC**: **미착수**.
- 세션 상한(10세션)에 도달해 자율 작업 종료.
- 오너 결정 대기 페이지 4건(004~007) 게시됨 — 답이 없어 진행이 막힌 상태.
- DB 마이그레이션·플래그 활성화 같은 **비가역 작업은 오너 승인 없이 진행 불가**라는 안전 규칙이 적용됨.
